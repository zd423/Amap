# 更新日志

本文档用于记录 AMap Companion 的人工整理更新日志。服务端同步 GitHub Release 后，还会生成面向客户端的更新日志：

```text
update_server/public/CHANGELOG.md
```

## 2026-05-19 巡航红绿灯 wrapper 与定制地图支持

- 支持解析定制高德地图广播的 `lightsData` JSON，巡航时可同时显示左转、直行等多个方向的红绿灯倒计时。
- 新增 `clearLights` / `EXTRA_CLEAR_LIGHTS` 清屏信号处理，离开路口或 wrapper 列表为空时可立即清除旧倒计时。
- 修正巡航 wrapper 状态映射：按实际测试将 `0` 归一化为红灯、`1/2/4` 归一化为绿灯，避免绿灯显示成红灯、红灯显示成黄灯。
- 调整巡航红绿灯保留时长，去除固定 45 秒保留，改为跟随实际倒计时自然归零，减少不在路口时残留显示。
- 新增开机自启、高德前台隐藏中控悬浮窗、导航/巡航退出隐藏仪表等显示策略选项。
- 文档和应用内开源信息新增定制地图 skill 仓库链接，并标注巡航多方向红绿灯需要配合定制地图。
- 文档和应用内开源信息新增定制地图 skill 与已改高德 APK 的镜像站链接，方便 GitHub 访问不稳定地区下载。
- 应用内定制地图 Skill 和已改高德下载入口改为各自一个按钮，点击后可选择“镜像站（快）”或“GitHub 原站（可能较慢）”，减少按钮数量并保留两条下载线路。
- 修正定制地图 Skill 镜像入口：镜像站改为直接下载仓库 ZIP，避免代理 GitHub 仓库页面时提示 `invalid input`。
- 修正显示策略不生效的问题：高德前台隐藏会引导开启“使用情况访问权限”，仪表退出隐藏不再被单纯道路名广播误判为导航活跃。
- 首页新增公告栏，显示反馈/交流群 QQ 群号 `1106923186`。
- 主背景透明度下限调整为 `0%`，设置为 `0%` 时外层背景和描边都完全透明。
- 目标应用选择保留 `com.autonavi.*` 包名优先逻辑，同时补充名称包含“地图”的应用；弹窗新增“显示所有应用”按钮，并优化带图标的选择列表样式。
- 新增“查看/保存日志”功能，可在应用内查看最近 `logcat` 调试信息，并支持复制或保存日志文件；优先保存到 `/sdcard/amap_log`，无权限时回退到应用私有日志目录，便于用户反馈和排查问题。

## 2026-05-05 副屏投屏选择与显示稳定性优化

- 新增“选择投屏屏幕”功能：当车机存在多块副屏或虚拟显示时，可手动指定悬浮窗镜像投到哪一块屏幕，也可继续使用自动选择。
- 副屏投屏窗口改为透明主题，减少白色底幕对仪表盘画面的遮挡，更贴近真正的悬浮显示效果。
- 优化副屏悬浮窗的布局定位逻辑：在副屏尺寸和面板尺寸确定后再校正位置，降低面板被旧坐标挤出屏幕外、看起来像“没有显示”的概率。
- 改进副屏镜像窗口的透明背景、边界计算和重新布局时机，提升多屏设备上的显示稳定性。
- 补充前台服务与通知初始化的兼容性处理，减少不同系统版本上副屏镜像启动过程中的异常差异。

## 2026-05-05 宽屏布局与透明度优化

- 主界面改为优先适配 `1920×720` 等车机宽屏分辨率，操作区与设置区拆分为双栏布局，浏览与点击更顺手。
- 主操作按钮、开源信息按钮和自定义显示选项在宽屏下改为双列排布，减少纵向滚动距离。
- 悬浮窗主背景由简单的“透明/不透明”切换改为“主背景透明度”滑动条，可按场景细调地图透出程度。
- 主背景透明度调整会同步作用于主悬浮窗、副屏镜像和应用内预览，所见即所得。
- 文字模式改为“自动模式”和“浅色模式”：自动模式会根据主背景透明度自动切换文字颜色，降低不同底图上的阅读压力。
